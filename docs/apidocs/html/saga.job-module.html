The SAGA Job API package.

The SAGA job API is used to submit and monitor jobs (computational tasks) on local and remote queueing systems and grid/distributed middleware.

More on the concepts of the SAGA API can be found here: http://www.gridforum.org/documents/GFD.90.pdf
